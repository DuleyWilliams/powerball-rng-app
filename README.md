# Powerball RNG Engine

A Powerball analysis and ticket generation tool built with React, Python, and Streamlit.

## Features

- Weighted random ticket generation
- Historical Powerball draw database
- Tennessee Powerball data updates
- Hot number analysis
- Cold number analysis
- Powerball frequency tracking
- Repeated pair analysis
- Ticket scoring engine
- Combinatorial condensation
- CSV export

## Technologies

### Frontend
- React
- Vite
- TailwindCSS

### Backend
- Python
- Streamlit
- Pandas
- Requests
- BeautifulSoup
